# 💹 나스닥 배당 투자 리서치 시스템

나스닥 상장 배당주를 대상으로 5가지 투자 전략을 분석하고, 투자금액에 맞는 맞춤형 포트폴리오를 추천하는 **Streamlit 인터랙티브 대시보드**입니다.

---

## 📸 대시보드 페이지 구성

| 페이지 | 기능 |
|--------|------|
| 💰 **내 투자 플래너** | 투자금액 입력 → 맞춤 전략 추천 + 미래 수익 시뮬레이션 |
| 📈 **배당 수입 예측** | 종목별 복리 성장 + 연도별 배당 수입 예측 |
| 🥧 **포트폴리오 배분** | 섹터/종목 배분 분석, Risk-Return 산점도 |
| 📅 **배당 캘린더** | 월별 배당 히트맵, 현금흐름 균등성 분석 |
| ⚔️ **전략 비교** | 5개 전략 백테스트 성과 비교 (2005~현재) |
| 🌱 **성장 & 지속가능성** | 배당 성장 이력, 지속가능성 점수, 배당함정 탐지 |
| 📞 **커버드콜 시뮬레이터** | Black-Scholes 기반 옵션 프리미엄 수입 계산 |

## 🚀 빠른 시작

### 1. 설치

```bash
git clone <repo-url>
cd futureAllocationSystem
pip install -r requirements.txt
```

### 2. 데이터 로딩 (최초 1회)

```bash
# 소규모 유니버스 (약 50종목, ~1분)
python setup_data.py

# 전체 유니버스 (약 137종목, ~2분)
python setup_data.py --full
```

### 3. 대시보드 실행

```bash
streamlit run dashboard/app.py
```

브라우저에서 `http://localhost:8501` 접속

## 💼 5가지 투자 전략

### 1. 월배당 최대화 (Max Monthly Dividend)
- 월배당 ETF/주식 중심 (JEPI, JEPQ, O, MAIN 등)
- 최고 현금흐름 최적화
- 스크리닝: 월 12회 이상 배당, 수익률 3%+, 시총 $500M+

### 2. 실버 연금 배당 (Silver Pension)
- 안정적 4~6% 수익률, 저변동성, 대형주
- 스크리닝: 시총 $10B+, 베타 1.0 미만, 10년+ 연속 배당, 수익률 4~7%

### 3. 장기 배당 성장 (Dividend Growth)
- 연 8% 이상 배당 성장률, 낮은 배당성향
- 스크리닝: 5년 CAGR 8%+, 배당성향 60% 미만, 5년+ 연속 배당 증가

### 4. 퀄리티 배당 코어 (Quality Core)
- ROE 15%+, 안정적 현금흐름, 10년+ 배당 이력
- 스크리닝: ROE 15%+, 양의 FCF, 10년+ 연속 배당

### 5. 커버드콜 배당 (Covered Call)
- 고배당 + 높은 옵션 유동성
- 배당수익 + 콜옵션 프리미엄 합산 수익률 최대화

## 📊 분석 기능

### 배당함정 탐지 (Yield Trap Detection)
지속 불가능한 고배당 종목을 6가지 레드플래그로 자동 탐지:
- 수익률 8%+ & 배당성향 80%+
- 주가 폭락으로 인한 수익률 상승 (1년 -30% 이하)
- 이익 감소 중 배당 유지
- 잉여현금흐름(FCF) < 배당 지급액
- 높은 부채비율 + 고배당
- 최근 3년 배당 삭감 이력

### 배당 지속가능성 점수 (Sustainability Score)
4가지 항목, 100점 만점:
- **배당성향** (25점): 배당성향이 낮을수록 고득점
- **FCF 커버리지** (25점): FCF/배당지급액 비율
- **이익 안정성** (25점): EPS 변동계수 기반
- **배당 추세** (25점): 5년 배당 CAGR

| 점수 | 등급 | 해석 |
|------|------|------|
| 80~100 | A | 매우 안전 |
| 65~79 | B | 안전 |
| 50~64 | C | 보통 |
| 35~49 | D | 주의 |
| 0~34 | F | 위험 |

### 배당락일 타이밍 분석
- 배당락 전후 평균 주가 패턴 분석
- 최적 매수 타이밍 (배당락 N일 전) 탐색
- 배당락일 평균 주가 하락폭 vs 배당금 비교

## 💰 내 투자 플래너 - 금액별 추천 전략

| 투자 금액 | 추천 전략 조합 | 예상 배당수익률 |
|-----------|---------------|----------------|
| $10,000 미만 (약 1,350만원 미만) | 월배당 ETF 중심 | 5.5~7% |
| $10,000~$50,000 (약 1,350만~6,750만원) | 월배당 40% + 실버연금 35% + 퀄리티 25% | 4.5~6% |
| $50,000~$100,000 (약 6,750만~1억3,500만원) | 4개 전략 균등 분산 | 4~5.5% |
| $100,000 이상 (약 1억3,500만원 이상) | 5개 전략 + 커버드콜 포함 | 4.5~6% |

## 📡 데이터 소스 및 한계

### 데이터 소스
- **yfinance** (무료): 가격 히스토리, 배당 이력, 현재 재무정보, 옵션 체인

### 캐시 전략
| 데이터 유형 | 저장 방식 | TTL |
|------------|-----------|-----|
| 가격 히스토리 | Parquet 파일 | 1일 |
| 티커 정보 (.info) | SQLite JSON | 1주 |
| 재무제표 | SQLite JSON | 1개월 |

### 알려진 한계
> ⚠️ **2021년 이전 백테스트**: ROE, FCF, 배당성향 등 펀더멘털 데이터가 제공되지 않아
> 가격·배당 기반 지표만 사용합니다. 결과가 실제보다 낙관적일 수 있습니다.

> ⚠️ **커버드콜 백테스트**: 과거 옵션 데이터가 없어 Black-Scholes + 히스토리컬 변동성으로
> 프리미엄을 추정합니다. 실제 시장 프리미엄과 차이가 있을 수 있습니다.

> ⚠️ **생존 편향**: 현재 거래 중인 종목을 기준으로 분석하므로 상장폐지·배당 삭감 종목이
> 제외되어 백테스트 결과가 과대 평가될 수 있습니다.

## ⚠️ 면책조항

이 시스템은 **교육 및 리서치 목적**으로 제작되었습니다.
모든 분석 결과는 과거 데이터 기반 추정치이며 미래 수익을 보장하지 않습니다.
실제 투자 결정은 반드시 전문 금융 전문가와 상담 후 진행하시기 바랍니다.
